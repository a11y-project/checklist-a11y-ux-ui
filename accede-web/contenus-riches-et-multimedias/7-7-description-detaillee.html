7.7. Prévoir un moyen d'accès à la description détaillée de chaque contenu riche

Accueil› Contenus riches et multimédias› 7.7
Un moyen d'accès à la description détaillée doit être prévu pour chaque contenu riche.

Un contenu riche véhicule tellement d'informations qu'une description textuelle parallèle devient nécessaire : infographie, graphique (diagramme, courbe…), carte géographique.

Modes d'accès à la description

  • Directement sur la même page, à proximité du contenu riche
  • Sur une autre page, accessible via un lien placé près du contenu riche

Exemples

Graphique

Un bouton « Afficher la version texte » permet d'accéder à la description détaillée d'un graphique circulaire.

Carte géographique

Un texte listant l'adresse, les stations de métro et les arrêts de bus les plus proches accompagne la carte.

Correspondance RGAA 4.1.2
