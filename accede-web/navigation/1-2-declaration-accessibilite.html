1.2. Prévoir une déclaration d'accessibilité

Accueil› Navigation› 1.2
Une déclaration d'accessibilité doit être prévue et atteignable depuis le même endroit sur toutes les pages du site.

Pour les sites soumis au RGAA (Référentiel Général d'Amélioration de l'Accessibilité), le lien vers la déclaration doit inclure une mention obligatoire reflétant le taux de conformité.

Libellé recommandé

« Accessibilité : partiellement conforme »

Les informations obligatoires pour cette déclaration sont disponibles sur le site du RGAA, section « Déclaration d'accessibilité ». Un modèle de déclaration est également proposé.

Correspondance RGAA 4.1.2
