AcceDe Web — 1. Navigation

1.1. Prévoir au moins deux moyens de navigation parmi un menu principal, un plan du site et un moteur de recherche

Accueil› Navigation› 1.1
Parmi les trois moyens de navigation suivants, au moins deux doivent être présents sur le site : un menu principal, un plan du site permettant de comprendre la structure complète, ou un moteur de recherche global couvrant l'ensemble des contenus.

Ces éléments de navigation doivent être disponibles depuis le même endroit sur toutes les pages du site.

Il est possible d'implémenter les trois moyens simultanément.

Exemples

Menu principal structuré

Un menu de navigation hiérarchique permettant d'accéder à toutes les grandes rubriques du site.

Plan du site

Une page dédiée montrant l'arborescence complète du site, accessible depuis un lien en pied de page.

Moteur de recherche amélioré

Un moteur de recherche avec filtres, case à cocher pour recherche exacte, et menu déroulant permettant de cibler une rubrique spécifique plutôt qu'une recherche globale.

Correspondance RGAA 4.1.2
