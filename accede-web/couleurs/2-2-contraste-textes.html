2.2. Assurer un contraste suffisant entre les textes et l'arrière-plan

Accueil› Couleurs› 2.2
Le contraste entre les textes et l'arrière-plan doit être suffisant. Ratio minimum : 4,5:1. Cette règle s'applique à tous les types de textes : textes standards, textes incrustés dans des images, dans des vidéos, etc.
La police utilisée n'affecte pas les exigences de contraste. La taille et l'épaisseur du texte peuvent théoriquement permettre une réduction du ratio, mais il est fortement recommandé de toujours respecter le contraste de 4,5 sur les textes.
Vigilance requise avec les dégradés, la transparence et les images d'arrière-plan.

Outils recommandés

  • Colour Contrast Analyser (CCA) : outil gratuit permettant de saisir manuellement les codes hexadécimaux ou d'utiliser une pipette pour extraire les couleurs, et d'afficher les résultats WCAG 2.1.
  • Contrast Finder : pour trouver des combinaisons de couleurs optimisées.
  • Extension Firefox WCAG Contrast Checker
Une charte graphique alternative suffisamment contrastée peut être proposée via un bouton "Augmenter le contraste" activable par l'utilisateur.

Correspondance RGAA 4.1.2

Critère : 3.2
