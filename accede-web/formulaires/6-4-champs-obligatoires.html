6.4. Indiquer clairement les champs obligatoires

Accueil› Formulaires› 6.4
Les champs obligatoires dans les formulaires doivent être clairement indiqués : un symbole (comme l'astérisque « * ») ou une mention textuelle telle que « (obligatoire) » doit apparaître dans le libellé de chaque champ obligatoire. Lorsqu'un symbole est utilisé, une légende placée au début du formulaire doit préciser sa signification (ex. : « * champs obligatoires »).

Cas particuliers

  • Tous les champs obligatoires : une seule mention « Tous les champs sont obligatoires. » en début de formulaire suffit.
  • Champ unique : lorsqu'un formulaire ne comporte qu'un seul champ, l'indication obligatoire n'est pas nécessaire selon le RGAA 11.10.
  • Majorité de champs obligatoires : on peut indiquer uniquement les champs facultatifs avec la mention « Tous les champs sont obligatoires sauf ceux indiqués comme facultatifs ».
L'astérisque doit avoir un contraste suffisant et être correctement intégré en HTML pour l'accessibilité.

Correspondance RGAA 4.1.2

Critère : 11.10
