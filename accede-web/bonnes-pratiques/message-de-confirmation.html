Prévoir un message de confirmation

Accueil›Bonnes pratiques
Chaque fois qu'un formulaire est validé avec succès, prévoir un message de confirmation rappelant la nature de l'action effectuée, apparaissant clairement à l'écran.
Les messages de confirmation ne doivent jamais disparaître automatiquement. Ils ne doivent se fermer que sur action manuelle (bouton de fermeture, nouvelle page, nouvel envoi formulaire, etc.).

Exceptions — message non nécessaire

  • Formulaire de connexion menant à une page profil (résultat évident)
  • Bouton « Étape suivante » dans un formulaire multi-étapes
  • Formulaire de commentaires affichant le commentaire posté
