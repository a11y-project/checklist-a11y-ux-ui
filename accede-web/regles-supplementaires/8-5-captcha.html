Prévoir une alternative pour chaque CAPTCHA uniquement visuel

Accueil› Règles supplémentaires
Pour chaque CAPTCHA uniquement visuel, une alternative doit être prévue.

Exemples d'alternatives

  • Une version audio du CAPTCHA (lien « Écouter le code »)
  • Une alternative textuelle sous forme de question simple ou mathématique
  • Un lien « Essayer un autre code » pour obtenir une nouvelle image
Il est vivement recommandé de proposer un moyen de recharger le CAPTCHA, car ils sont souvent difficiles à déchiffrer du premier essai.
Proposer en complément un moyen de contacter le responsable du site (formulaire contact, numéro téléphonique) pour les cas où le CAPTCHA reste inaccessible.
Le reCAPTCHA v2 de Google n'est pas considéré conforme et pose des difficultés pour les utilisateurs clavier et lecteur d'écran. Alternatives recommandées : Friendly Captcha, reCAPTCHA v3 (ne nécessite aucune action utilisateur).

Correspondance RGAA 4.1.2
