Prévoir un moyen pour contrôler les sons déclenchés automatiquement

Accueil› Règles supplémentaires
Un moyen de contrôler les sons déclenchés automatiquement doit être prévu.
Le RGAA prévoit une exception : les sons automatiques sont tolérés s'ils ne dépassent pas trois secondes de durée.

Correspondance RGAA 4.1.2

Critère : 4.10
