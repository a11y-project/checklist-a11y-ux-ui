AcceDe Web — 8. Règles supplémentaires

Prévoir des intitulés identiques pour les champs dont la fonction est identique

Accueil› Règles supplémentaires
Lorsque des champs dont la fonction est identique sont proposés plusieurs fois sur le site, leurs intitulés doivent également être identiques.

Exemple

Formulaire de réservation multi-onglets

Dans un formulaire avec les onglets Train, Vol, Hôtel, Voiture, Séjour, les champs « Départ * » et « Arrivée * » conservent les mêmes intitulés dans tous les onglets.

Non conforme

Utiliser « Arrivée * » dans un onglet et « Destination * » dans un autre pour le même type de champ.

Correspondance RGAA 4.1.2

Critère : 11.3
